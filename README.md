# new-project2

A simple interactive dashboard built with [Dash](https://dash.plotly.com/) and [Plotly](https://plotly.com/python/). It displays a sample bar chart from a small pandas DataFrame.

## Features

- Interactive bar chart powered by Plotly Express
- Web-based UI served by Dash
- Sample categorical data (Categories A–D with numeric values)

## Prerequisites

- Python 3.8 or newer

## Installation

Install the required packages:

```bash
pip install dash plotly pandas
```

## Usage

Run the dashboard:

```bash
python testtt.py
```

Open your browser and go to [http://127.0.0.1:8050](http://127.0.0.1:8050).

The app runs in debug mode by default, so code changes will reload automatically.
